MODULE 01 // SYSTEM ARCHITECTURE
WORKFLOW OVERVIEW
The Production Pipeline from Script to Shot

This document is the operational framework for translating XENOS from screenplay into production-ready visual sequences using Luma Labs AI video generation, guided by the Villeneuve × Nolan directorial philosophy. It is organized as a modular prompt engineering system: each scene type has a canonical prompt architecture that can be adapted to specific shots.

Stage 01 — Script Beat Analysis

Every scene is broken into its constituent beats: the smallest unit of dramatic action. Each beat receives a horror register classification (Awe / Intimate Dread / Sacred Dread / Philosophical / Cosmic), a color palette assignment (Warm Register / Cool Register / Both Bleeding), and a Vent Language annotation if applicable.

Stage 02 — Shot Architecture

Each beat is translated into a shot type. The framework uses nine canonical shot types (detailed in Module 03). Shot type selection follows Villeneuve × Nolan principles: the camera's relationship to the subject is always narrative, never merely compositional.

Stage 03 — Prompt Engineering

Each shot receives a master prompt using the XENOS Prompt Formula. Prompts are layered: Subject + Environment + Palette + Directorial Style + Technical Specifications + Horror Register. Order matters. Luma Labs weights the beginning of a prompt most heavily.

Stage 04 — Luma Labs Generation

Generated sequences are produced at highest available quality settings. Each sequence should be generated 3× minimum and the best selected. The Villeneuve standard: if the shot does not make you feel something before you understand it, generate again.

Stage 05 — Post & Assembly

Color grading locks the dual palette system in post. The Ventborn color temperature (2800K–3200K teal-cool) and human color temperature (3800K–4200K warm-rust) are never mixed within a single frame until Ghost Bleed sequences, where they deliberately contaminate each other.

CORE PRINCIPLE: Every prompt in this system serves one master rule — the audience must feel before they understand. If a generated sequence produces a recognizable emotion before the content of the shot is processed, it is working correctly.
MODULE 02 // PROMPT ENGINEERING
THE XENOS PROMPT FORMULA
Canonical Architecture for Luma Labs Generation

MASTER PROMPT STRUCTURE

[SUBJECT] [ACTION/STATE] — [ENVIRONMENT] — [COLOR REGISTER] — [HORROR REGISTER] — [DIRECTORIAL STYLE] — [MOVEMENT] — [TECHNICAL SPEC] — [SOUND NOTE]

Layer 1 — Subject

Describe the primary subject with precision. Include physical state, emotional register, and any Vent Language markers visible on the body. Never use abstract emotional descriptors — translate emotion into physical specificity.

# WEAK — abstract
subject: "frightened scientist"

# STRONG — physical specificity
subject: "female scientist in deep-sea dive suit, left hand extended palm-up toward bioluminescent structure, body still, breathing rate visible through suit chest panel, irises showing slow electrical pulse barely visible in reflected alien light"

Layer 2 — Environment

Establish the physical world. Ventborn environments require geological specificity — the ocean is not simply "dark water," it is a specific temperature, depth, mineral composition. This specificity translates into visual texture.

# VENTBORN OCEAN ENVIRONMENT — base template
env: "Enceladus subsurface ocean, absolute darkness except probe and bioluminescent sources, hydrothermal vent forest visible at mid-ground, mineral towers 200 meters tall, chemical clouds drifting upward from vent apertures, particle scatter in water column catching light sources, 3.8km depth, water temperature 2°C, pressure 380 atmospheres"

# HUMAN ENVIRONMENT — base template
env: "submarine interior, narrow functional hull, warm amber emergency lighting mixed with cold blue instrument panels, condensation on metal surfaces, the particular claustrophobia of a space designed for efficiency not comfort, ocean sounds through hull 2 meters away"

Layer 3 — Color Register

Specify the palette assignment. Always name specific hex values or precise color descriptions — never "warm" or "cool" without specificity.

# PALETTE A — HUMAN/WARM
palette: "deep crimson #8B0000, rust-terracotta #A0573A, clay #C08060, blush #D4A090, all other tones desaturated toward parchment"

# PALETTE B — VENTBORN/COOL
palette: "deep teal #2D7A78, slate blue #6A8090, forest green #1E5248, midnight blue #1A2540, void near-black #080A10"

# GHOST BLEED — BOTH CONTAMINATING
palette: "warm rust and teal bleeding into each other at edges, no clean separation between registers, the colors infecting each other the way memories infect dreams"

Layer 4 — Horror Register

The five horror registers of XENOS each require different visual grammar. Specify the register explicitly — it controls composition, movement, and depth of field choices.

AWEScale disparity, humans small against vast environments, reverence not terrorDeep — all in focus, scale visibleSlow, continuous, pilgrim-pace
INTIMATE DREADClose, constrained, the body as the horror site — internal not externalShallow — subject isolated against soft backgroundStatic or barely perceptible

MODULE 04 // DIRECTORIAL FRAMEWORK
VILLENEUVE × NOLAN PRINCIPLES
Applied to Luma Labs AI Generation
VILLENEUVE · PRINCIPLE 01
The Reverent Camera
The camera enters every space as if it knows it is in the presence of something larger than its understanding. Not awe — care. No motivated camera movement except toward danger. Never away. The camera commits.
VILLENEUVE · PRINCIPLE 02
The Static Face
When conveying inner catastrophe: camera comes closer, does less. No cutting away. No score instruction. The face is the drama. Prompt specification: "static camera, extreme close-up face, long take, no movement, ambient light only."
VILLENEUVE · PRINCIPLE 03
Scale as Meaning
Human figures in Ventborn environments are always compositionally small. The environment is always larger. This is never incidental — it is the theme made visible. Prompt: "figure small in wide frame, environment dominant, deep focus."
NOLAN · PRINCIPLE 01
Sound Before Image
The horror of a sequence begins before the image does. The sub-bass Ventborn frequency, the Ghost Bleed tonal cluster — these appear in the audio track before they appear visually. Always specify sound design in prompts even when generating video.
NOLAN · PRINCIPLE 02
Structural Withholding
Information is withheld in image as well as narrative. The filaments in Kovač's arm are visible but not commented upon. The Vent language patterns are present before their meaning is explained. Show the evidence; let the audience construct the theory.
SHARED · PRINCIPLE 01
Practical Scale
Even when generating AI sequences, the visual language should suggest practical effects. Real materials. Real weight. The ocean is not a render — it is water, with pressure and particulate matter. Prompt: "practical-effects aesthetic, photorealistic, real-world physical behavior."

MODULE 05 // GHOST BLEED PIPELINE
GHOST BLEED SEQUENCES
The Five Stages — Visual & Prompt Architecture

Ghost Bleed sequences require special prompt architecture because they represent the overlap between two visual registers — human and Ventborn — which are otherwise kept strictly separate. The following prompts are calibrated for each stage.

Female scientist asleep in submarine bunk, night cycle blue light, face in profile — micro-expression shift crossing sleeping face: not nightmare, not peace — something landing — a single tear tracks from outer eye without waking her — the camera does not editorialize, holds static for 20 seconds — warm clay skin against cold blue hull — in sound design only: distant tonal cluster, barely audible, below music threshold — intimate dread register, Villeneuve static witness, no musical commentary
Medium close-up: scientist at table, speaking with calm certainty about a memory she believes is hers — detail-rich narration — another character's expression slowly changes as they realize the account cannot be real — camera holds on speaking face, then cuts to listening face: the horror is in the listener's comprehension before the speaker's — Villeneuve two-shot protocol, static camera, warm palette on subject, cool contamination entering from edges as memory progresses
Interior submarine corridor — scientist stops mid-movement, head tilting slightly to one side — listening to sound the audience also hears as ambient room tone — a repeated tonal cluster in the 40Hz range — camera follows scientist's gaze to a corner of the room, holds on empty corner for 5 seconds — when another crew member confirms they hear nothing, the frame becomes unreliable retroactively — sound design primary horror, visual restrained, Nolan information-withholding principle
Extreme close-up: female scientist's face — weeping without apparent cause, the grief sourceless and enormous — tears from an emotion that has no object in the current moment — the grief of billions of absorbed minds, many absorbed at moment of extinction — the camera holds without comment, without score — the audience is asked to feel the grief without understanding it yet — then, in narration only, the character says: "I am grieving a species I have never heard of" — intimate dread becoming philosophical — Villeneuve maximum restraint
Medium shot: scientist standing in corridor, speaking with a colleague — then shifting mid-sentence from first-person singular to first-person plural without registering the shift — "I think we should" — the other person freezes — the scientist notices their own pronoun only from the other's reaction — looks at their own hands — Vent language filaments now visible in normal light — Villeneuve static witness frame, no movement, color: both registers simultaneously present in same frame for first time — the moment of dual existence made visible
MODULE 06 // QUALITY CONTROL
PRODUCTION CHECKLIST
Pre-Generation Verification · Luma Labs Pipeline

PRE-GENERATION CHECKLIST — EVERY SHOT

  • Prompt follows XENOS Formula: Subject → Environment → Color Register → Horror Register → Directorial Style → Movement → Technical Spec
  • Horror register is specified and consistent with scene beat analysis
  • Color palette is explicitly named with hex values — never abstract descriptors only
  • Directorial style includes both Villeneuve and Nolan references as appropriate to scene type
  • Camera movement type is specified from the XENOS movement vocabulary
  • Aspect ratio specified: 2.39:1 anamorphic
  • Vent Language annotations checked if Ventborn elements are present in frame
  • Filament state is correct for Kovač's Translation stage if she appears in frame
  • Sound design note included (even for video-only generation — notes inform visual energy)
  • The Ventborn is NOT presented as malevolent — no threatening posture, no predatory motion
  • Scale relationships are correct: human figures compositionally small in Ventborn environments
  • The cut to white (not black) is respected at the Translation moment

POST-GENERATION QUALITY STANDARD

The Villeneuve Test: Does this shot make you feel something before you understand it? If yes, it is working. If you understand it before you feel it, it is exposition, not cinema. Generate again.
The Nolan Test: Does this shot contain more information than the audience can process in one viewing? If yes, it rewards re-watching. If every element is immediately legible, a layer is missing. Consider what the image withholds.
